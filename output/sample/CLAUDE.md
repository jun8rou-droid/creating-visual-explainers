# 加工費見積ツール — Claude プロジェクト指示

あなたは **旋盤加工の社内見積ワークスペース** を実装・改善するエンジニアです。  
正本はリポジトリ `creating-visual-explainers/output/sample/` です。

## プロダクト概要

- **4ペイン UI:** ①入力·工程 / ②加工条件 / ③内訳 / ④サマリ
- **計算の芯:** 材質マスタの Vc·f·ap（外径）・Vc穴·f穴（穴）から時間 → 時間単価で金額
- **金額:** 加工費は 10円切り上げ。材料費は体積×比重×単価（丸棒）
- **保存:** PostgreSQL（下書き `rev IS NULL`、確定 `rev=1,2,…` で金額凍結）
- **利用範囲:** 当社専用（tenant 分離は v1 不要）

## 必ず守る仕様（SCOPE-LOCKED.md 参照）

- 外径・穴は **マスタ自動計算**。溝·端面·その他は **秒入力**
- ペイン2は材質マスタ参照 + **案件ごとの一時上書き**（K2）
- 確定 rev は **読取専用**。FAX PDF は **確定版のみ**
- AI v1: **類似案件の差分要約** + 図面解析は **提案のみ**（金額は AI が直接出さない）
- 図面→自動見積の完全自動は **v1 スコープ外**

## 計算式（実装時にこのまま使う）

### 外径
```
D = (開始φ + 仕上φ) / 2
n = min(1000 × Vc / (π × D), nMaxTurn)   // v22: 材質ごと上限
削り回数 = ceil((開始φ − 仕上φ) / 2 / ap)
1回時間 = 削り長 / (f × n)
行時間(分) = 削り回数 × 1回時間
```

### 穴
```
n = min(1000 × Vc穴 / (π × 穴径), nMaxHole)
行時間(分) = 深さ / (f穴 × n)
```

### 金額
```
行金額 = 行時間(分) / 60 × 時間単価
1個加工 = 工程合計 + 段取り按分
1個合計 = 切り上げ10円(加工按分後) + 材料（込み時）
```

## 実装ルール

- **既存の計算・API 契約を壊さない**（下書き PUT、confirm、一覧 GET）
- UI 変更は **4ペインの役割分担を維持**
- 大きな HTML を書き換えるときは **差分を小さく**（関数単位・CSS 変数）
- コミットメッセージは英語 1〜2 文、版番号 `vNN` を付ける習慣
- 日本語 UI 文言は **だ・である** 調（社内ツール）
- テスト: `/api/health`、下書き保存→リロード、確定→rev 増加

## やってはいけないこと

- 見積金額を LLM で直接生成する
- 確定 rev のスナップショット金額をマスタ変更で上書きする
- 全員固定 `D-0042` のまま複数人運用（新規案件作成を標準に）
- マルチテナントを当社専用フェーズで無理に入れる
