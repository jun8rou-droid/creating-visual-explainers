取り込み・実行

CLI の --csv --week などに相当。本番は毎週月曜 08:00（JST）に CSV が自動送信され、受信がトリガー。データ週は月 08:00:00〜翌週月 07:59:59。

相当するコマンド

python -m sales_report_automation \
  --csv sample/week_sample.csv \
  --week 2026-W14 \
  --dry-run
