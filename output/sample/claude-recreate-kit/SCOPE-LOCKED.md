# 確定仕様（変更する場合はユーザー確認必須）

`machining-quote-design-memo.md` の要約。Claude は **ここに書かれた決定を勝手に変えない**。

## 画面

- **4ペイン:** ①入力（上:案件 / 下:工程リスト+入力）②加工条件 ③内訳 ④サマリ
- 工程追加は **対話式**（タイプ選択 → 必要項目のみ）
- 外径: 開始φ(任意)→仕上φ→削り長 / 穴: φ→深さ / その他系: 秒

## 計算

- 外径・穴: **マスタ自動**（T2, Mα）
- 溝·端面·その他: **秒入力**
- 加工費: **10円切り上げ**
- 材料: 丸棒 体積×比重×¥/kg（L は入力そのまま）
- 段取り: 10分刻み、本数按分

## 材料モード

- **材料込み** / **材料支給**（ヘッダー切替）
- 支給時は材料費 0、比較は加工費ベース

## 版・保存

- **下書き** (`rev IS NULL`): 入力保存、金額は都度再計算
- **版を確定**: 金額・メモ3項目を **凍結**、下書き行削除
- 確定 rev 表示は **読取専用**（P2）
- 正式 ID: 初回確定で `Q-YYYYMMDD-###`、それまでは `D-####`

## マスタ

- 材質: Vc, f, ap, Vc穴, f穴, 比重, ¥/kg（実装: + nMaxTurn, nMaxHole）
- 品名: catalog / special / other
- 共通: 時間単価、社名、TEL、FAX、有効日数(14)
- 回答テンプレ: 込み用・支給用 2種

## 類似案件（Hα）

- 条件: 材質一致 · φ一致 · L±5mm · 最大5件（緩めない）
- タグ: 材質·φ·L·工程数（一致時のみ）
- 自動コピーなし

## AI（v1）

- **金額はマスタ計算のみ**（AI は出さない）
- 図面: 欄提案 + 工程提案（採用/却下）
- 類似: **差分要約** のみ本番スコープ
- API キーはサーバ側のみ

## FAX PDF（CT-FAX）

- **確定 rev のみ**
- 工程は載せない（品名·材質·φ·L·本数）
- PDF は DB に保存しない（都度生成）
- 担当者名なし

## データ保管

- **正本は PostgreSQL のみ**（NAS JSON 案は廃止）
- 当社専用（2026-07）: tenant 不要
- 見積は **残す**

## v1 でやらない

至急フラグ、開始位置mm、案件上書き永続化(K2)、社内メモ手書き、図面のみで見積完成
